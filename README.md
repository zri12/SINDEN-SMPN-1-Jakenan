# SINDEN

SINDEN adalah Sistem Informasi Digital Evaluasi Nilai untuk SMP Negeri 1 Jakenan.

## Stack

- React + Vite
- TypeScript
- Tailwind CSS
- Supabase
- Vercel

## Menjalankan Project

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Untuk tahap ini UI menggunakan data dummy dan struktur Supabase masih berupa placeholder. Folder `docs/` dan `REFERENSI UI/` dipertahankan sebagai dokumentasi dan referensi desain.
